# 🐌 Мудрый тритон — почтальон Лукоморья

Программа для определения минимального количества извинений, которые придется принести почтальону Тритону при доставке писем птицам, гнездящимся на ветвях дуба.

## 📖 Описание задачи

Мудрый тритон работает почтальоном в Лукоморье. Он доставляет корреспонденцию птицам, гнезда которых расположены на ветвях дуба. Каждое гнездо имеет уникальный номер (гнездо на первой развилке ствола имеет номер 1).

Тритон может передвигаться только по веткам и стволу (прыжки и перелеты исключены). Если тритон проходит через гнездо, но его обитатели не получают при этом корреспонденции, он вынужден извиняться. Причем если он проходит через одно и то же гнездо несколько раз, он извиняется каждый раз.

**Задача:** определить минимальное количество извинений, чтобы доставить все письма.

## 🧮 Алгоритм

Программа использует двухфазный рекурсивный алгоритм обхода дерева в глубину (DFS):

1. **Фаза 1 (`calc_letters`)** — вычисление общего количества писем в каждом поддереве (`branch_letters`).
2. **Фаза 2 (`pardon_count`)** — рекурсивный подсчет извинений:
   - При входе в гнездо без писем → +1 извинение
   - При наличии писем → забираем одно письмо
   - При возврате из ребенка без писем → +1 извинение
   - При наличии писем → забираем еще одно

Ветки, в которых нет писем (`branch_letters <= 0`), не посещаются — это экономит извинения.
